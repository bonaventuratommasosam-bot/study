Installation

Quick Start (2 minutes)

You need Hermes Agent (free). No API key needed — Gemini shared key included. Just install and go.

# Clone the repo git clone https://github.com/bonaventuratommasosam-bot/study.git cd study # Linux / Mac: bash setup.sh # Windows: python setup.py # Launch Study (terminal) hermes profile use study hermes chat # Or launch the web dashboard hermes gateway start → Open http://localhost:9119

Answer a few questions (name, country, school, LLM provider) and Study is live. Then just type: "Hey Study, I need to prepare for my exam. I have 30 days."

Questions

FAQ

Do I need to know how to code?

No. The setup script does everything. If you can use a terminal, you can install Study.

Can I use it for any exam?

Yes. Study adapts to your school system. If yours is missing, you can add it — it's open source.

Does it work on Windows?

Yes. Use python setup.py. Works on Windows, Mac, and Linux.

Is it really free? What's the catch?

No catch. Study is MIT licensed. It comes with a shared Gemini API key — zero setup, zero cost. For heavier use, you can bring your own key (Gemini, Groq, DeepSeek ~$3-5/month). Study runs locally on your machine — no HermesBro servers involved.

Why is it free when other HermesBro products are paid?

Education is a right. Tommy built Study while preparing for his own exams and decided to give it to the world. Other HermesBro products are professional tools for businesses.
