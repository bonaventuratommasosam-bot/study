## How to Propose a New Country

1. Open a GitHub Issue titled: `[New Country] Country Name`
2. Attach the official curriculum (PDF or link) if you have it
3. A maintainer will add it or guide you through doing it yourself

See [CONTRIBUTING.md](../CONTRIBUTING.md) for technical details.
